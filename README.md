# 🌌 Modulattice

**Watch modules weave themselves into existence** from pure descriptive potentials. An AI module generator that assembles C# code before your eyes. Built for Unity video game developers who are tired of writing boilerplate code and want to focus on design.

## Quickstart (30 seconds)

Run pip install fastapi uvicorn[standard] aiofiles ollama websockets to setup all the python dependencies.
use start.bat to launch the app and interface.


## Live Web Interface

1. Visit [modulattice.com](https://modulattice.com)
2. Type what you want: *"A player controller for a top-down view"*
3. **Watch it weave** → complete codebase appears in real-time
4. Copy-paste → production-ready

## Core Architecture

```
Modulattice("llama3-custom")
├── Parser -> Natural language -> AST
├── Weaver -> AST -> Multi-file architecture  
├── Compiler -> Architecture -> Production code
└── Visualizer -> Live progress bars
```

## Installation

```bash
pip install modulattice
```

**Supported LLMs:**
- `llama3-custom` (default)
- `gpt-4o`
- `claude-3.5-sonnet`
- **Bring your own** (future feature)


## Example Outputs

```
"A player health module that will handle players taking damage and dying"

-> 5 files generated:
├── audit.jsonl (an audit of what the system was thinking)
├── design.txt (a design document for the module)
├── Config.cs (a data class for the module if used)
├── PlayerHealth.cs (the module code file)
└── README.md
```
